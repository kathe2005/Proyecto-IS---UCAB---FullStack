Diagrama de Clases (Backend)

Ubicación: `docs/class-diagram`

Contenido:
- `index.html`: página con SVG que muestra un diagrama de clases simplificado del backend.
- `styles.css`: estilos para la visualización.

Cómo ver:
1. Abrir `docs/class-diagram/index.html` en tu navegador (doble-clic o `File -> Open`).
2. También puedes abrirlo desde VS Code con la extensión Live Server.

Notas:
- Este diagrama es una representación de alto nivel que ilustra los principales modelos, servicios y repositories. No cubre todas las clases auxiliares y adaptadores.
- Si quieres, puedo mejorar el diagrama (añadir más clases, mostrar herencias o generar automáticamente a partir del código fuente).
